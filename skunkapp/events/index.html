Welcome to SkunkApp Events

A platform for fun events related to gaming, programming, and more.


How to Create an Event

1. Use the Slash Command: Type /create-event in Discord.

2. Fill out the fields: Name, Description, Start Time, and End Time.

3. Review the SkunkApp's Events Guidelines.

4. Send the message to create the event.

5. If you added a thumbnail, it will be reviewed according to SkunkApp's Events Guidelines.

Your event will be created! Enjoy participating in SkunkApp events.

How to Update a Thumbnail

1. Use the Slash Command: Type /update-thumbnail in Discord.

2. Provide the event name and the new thumbnail attachment.

3. Wait for approval based on SkunkApp's Events Guidelines.

4. If approved, the thumbnail will be updated for the event.

Your event thumbnail has been updated! Enjoy the new look for your event.

Why Was the Event Deleted?

Event deletion is part of moderation to ensure adherence to SkunkApp's Events Guidelines.

Why an Event is Moderated: If an event is moderated, it means that a thumbnail or content might have been rejected for violating SkunkApp's Events Guidelines.

If I End an Event: Ending an event will delete it, and this action cannot be undone. End events only when you are certain you no longer need them.

Your event’s thumbnail will be updated once approved. Enjoy the events!

How to Search for Events

1. Use the Slash Command: Type /events in Discord.

2. Select the "Search" option.

3. Enter your search terms or criteria.

4. Send the message to execute the search command.

You can now easily find events! Enjoy exploring SkunkApp’s events.
